# 📚 BookShelf — Book Management System

A React + Tailwind CSS Book Management app with full CRUD operations.

## Features
- View all books in a responsive grid
- Add, Edit, Delete books
- Search by title or author
- Filter by genre
- Mock API integration (MockAPI.io)
- Toast notifications
- Mobile responsive

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- Axios
- React Router v6
- React Hot Toast
- MockAPI.io

## Setup Instructions

1. Clone the repo
   git clone <your-repo-url>
   cd book-management

2. Install dependencies
   npm install

3. Create .env file
   VITE_API_URL=https://your-mockapi-id.mockapi.io/api/v1

4. Run locally
   npm run dev
